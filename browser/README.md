# Browser Skills And Tools

Browser tools get their own space because they package reusable browser behavior, extension code, and installation instructions.

## Included Kits

- [video-speed-chrome-extension](kits/video-speed-chrome-extension): a Chrome extension that adds a draggable, hover-to-reveal controller with preset speeds for HTML5 video.

This is a Git submodule, so it keeps its own repository history while also appearing inside this hub.

## Pull The Kit Locally

```bash
git submodule update --init --recursive
```
