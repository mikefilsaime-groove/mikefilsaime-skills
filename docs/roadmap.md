# Roadmap

This hub starts as a curated public index. Over time, it can grow into a more complete library of installable skills and workflow kits.

## Near Term

- Add fuller examples for AEO, SEO, and mobile responsive workflows.
- Add a simple install section for skills that can be copied into local agent skill folders.
- Add screenshots or sample outputs where useful.
- Add tags to make the catalog easier to scan.

## Later

- Promote larger skills into standalone repositories.
- Add versioning guidance for public skills.
- Add short demo videos or walkthroughs.
- Add a compatibility matrix for Codex, Claude Code, and other agent environments.
