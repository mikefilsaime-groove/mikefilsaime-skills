## Publishing Pattern

Each skill should answer:

- What problem does this solve?
- When should someone use it?
- What inputs does it need?
- What workflow does it follow?
- What output should it produce?
- What guardrails keep the work useful and safe?

Use [the skill README template](../templates/SKILL-README-template.md) when adding a new public skill.
