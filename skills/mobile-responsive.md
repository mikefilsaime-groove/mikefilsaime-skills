# Mobile Responsive

The mobile responsive skill helps transform front-end pages into mobile-friendly, responsive, high-performing experiences.

## Use Cases

- Fix mobile layouts that overflow, clip, crowd, or break.
- Improve responsive behavior across phone, tablet, and desktop widths.
- Optimize touch targets, mobile navigation, typography, images, and layout stability.
- Improve mobile PageSpeed and Lighthouse outcomes.

## What It Looks At

- Mobile-first layout structure.
- Viewport configuration.
- Fluid typography and readable body text.
- Touch target sizing and spacing.
- Flexible media and responsive images.
- No horizontal scroll.
- Grid, flexbox, and component patterns.
- Mobile navigation.
- Mobile performance and visual QA.
